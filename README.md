# advice-generator-app-main

## Ejemplo en vivo

## Descripción 📑
Desafio de [Frontend Mentor](https://www.frontendmentor.io/challenges/advice-generator-app-QdUG-13db) realizado con react y tailwind.

## ¿Qué he aprendido en este proyecto? 🙇🏻 
El uso de useState, useEffect y consumir la api de https://api.adviceslip.com/ con Fetch.
